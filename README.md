*   **Mecánica de Stacks ("Unleash"):** Cada auto-ataque contra campeones te otorga Stacks de *Unleash*.

*   *Táctica:* Nunca gastes la Q contra minions a menos que necesites el Waveclear absoluto. ¡Guárdalo para que el Splash Damage rebote en toda la botlane!
